2026-07-16 — --full-edges: remove the decoder's ground-truth future-topology leak
==================================================================================

WHAT WAS DONE
New flag --full-edges (default False = exact baseline): the decoder's per-step
graph is a complete graph with self-loops over ALL sample agents, generated in
TrajectoryPredictionDataset.get() from the node count and substituted into the
existing tar_edge_index slot. This removes the last future-information leak
(which agents are present at future timesteps came from ground truth).
Verified: generated graphs byte-identical to preprocessing's (never previously
loaded) full_edge_idx files on 100/100 test samples.

Design (after 13-agent adversarial review):
- Generate instead of loading full_edge_idx files -> no extra disk I/O, no
  hard dependency on re-preprocessed data dirs (fixes highD --sparse crash),
  no doubled index traffic (substitution vs second Data attribute).
- "FE" tag in checkpoint names (train.py/test.py/visualize_predictions.py).
- Composes with --dynamic-edges (DEFE): distances recomputed from predictions
  over the full graph = fully deployable decoder, zero future information.
- Review fixes: FE tag + full_edges threading + first-error reporting in the
  visualization scripts (were crashing/opaque for FE checkpoints).

RESULTS (seed 1234; baseline 1.006/2.835/-0.108 val_nll -1.27,
         dynedge 1.036/2.946/-0.197 val_nll -3.63)
- FULL    : test ADE 1.048  FDE 2.968  ANLL -0.089  MR 0.419  val_nll -0.67
            (best val FDE of all runs: 2.756)
- FULLDYN : test ADE 1.050  FDE 2.978  ANLL -0.131  MR 0.433  val_nll -1.69
Conclusions:
1. Leak removal costs <=0.04 m ADE (edge of +/-0.03 seed band) - the paper's
   point metrics were essentially honest despite the topology leak.
2. Calibration pays for the leak fix; dynamic edge features recover most of
   it (val NLL -0.67 -> -1.69) - phantom-agent hypothesis supported: only
   the distance kernel can down-weight departed agents in a full graph.
3. Deployable config = --full-edges --dynamic-edges. Figure:
   results/comparison_leakfix.png; report section 4 updated.
